# 손을 주시오, 혈을 잡겠오 웹앱 서비스 구현 작업

## Setup
1. XAMPP(Control Panel)를 설치/실행 후 Apache와 Mysql을 실행.
    - Mysql 실행거부인 경우: cmd에서 services.msc를 치고 MySql 찾아서 실행중단 시킨 후 다시 실행
2. 콘솔에서 mysqlloginreg.py 실행
3. Repository에 들어와서 `npm start` 입력.
